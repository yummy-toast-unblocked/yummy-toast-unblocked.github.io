      Instructions

      Instructions:
      * Players line up facing a starting line.
      * On a signal, the first player runs forward as far as possible.
      * Once the player stops running, they mark their spot.
      * The next player runs from the starting line to beyond the previous player's mark.
      * This continues until all players have run.
      * The player who runs the farthest distance wins.
      FAQs:
      * Can I run past the previous player's mark? Yes, you can run as far as possible.
      * What happens if I stop running before reaching my farthest point? Your mark will be where you stopped.
      * Can I start running before the signal? No, you must wait for the signal to start running.
      * What happens if I cross the starting line before the signal? You will be disqualified.
      * Can I use any equipment to enhance my running? No, you must run without any external assistance.
