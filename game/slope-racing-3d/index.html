      Instructions

      Objective:
      Achieve the highest score possible by stacking blocks without them falling.

      Setup:
      1. Place the game board on a flat surface.
      2. Ensure all blocks are within reach.

      How to Play:
      1. Each player takes turns stacking one block onto the tower.
      2. Use only one hand to place the block.
      3. The block must be placed on top of the last block added, not off to the side.
      4. If the tower collapses during your turn, you lose the game.
      5. The game continues until the tower collapses or all blocks have been used.

      Winning:
      - The player who successfully places the most blocks before the tower collapses is the winner. If multiple players tie, the game can be replayed or the tied players can share the victory.

      FAQs:

      Q: How many players can play?
      A: The game can be played by 2 or more players.

      Q: What if the tower falls when it's not my turn?
      A: The player whose turn it is when the tower falls loses the game.

      Q: Can I use both hands to place a block?
      A: No, you must use only one hand to place the block.

      Q: What happens if I run out of blocks?
      A: If all blocks are used and the tower hasn't collapsed, the game ends. The player who placed the last block wins.

      Q: Can I adjust the blocks once placed?
      A: No, once a block is placed, it cannot be moved or adjusted.

      Q: Is there a time limit for placing a block?
      A: Typically, no time limit is enforced, but players should aim to be prompt to keep the game moving.

      Q: What are the dimensions of the blocks?
