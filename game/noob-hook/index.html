      Instructions

      Objective:
      The objective of the game is to test and improve your reflexes and hand-eye coordination by catching falling objects within a set time limit.

      Setup:
      1. Place the game board on a flat surface.
      2. Ensure all players have a clear view and access to the game board.
      3. Each player selects a character or marker.

      How to Play:
      1. Turn on the game device or start the timer.
      2. Objects will begin to fall from the top of the game board.
      3. Players must use their character or marker to catch as many falling objects as possible.
      4. Move quickly to catch the objects before they reach the bottom of the board.
      5. The game ends when the timer runs out.

      Scoring:
      - Each caught object awards points.
      - The player with the most points at the end of the game wins.
