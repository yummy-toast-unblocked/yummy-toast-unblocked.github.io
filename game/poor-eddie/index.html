      Instructions

      Gameplay:
      1. Connect adjacent tiles of the same color to create chains.
      2. Chains of three or more tiles are removed, allowing new tiles to fall.
      3. Match four or more tiles to create special tiles:
      - Vertical Striped Tile: Destroys all tiles in its column.
      - Horizontal Striped Tile: Destroys all tiles in its row.
      - Wrapped Tile: Destroys all tiles of the same color.
      4. Use special tiles strategically to clear large areas and earn bonus points.
      5. Complete the level by clearing all tiles within the time limit.
      FAQs:
      - How do I get more special tiles? Match more tiles to create larger chains.
      - Can I swap tiles? No, you cannot swap tiles in this game.
      - What happens if I run out of time? The level will fail, and you will need to restart.
      - How do I earn bonus points? Clear tiles quickly and use special tiles effectively.
      - Is there a limit to how many levels there are? No, there are endless levels to play.
