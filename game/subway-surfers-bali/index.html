      Instructions

      Objective: The objective of the game is to score as many points as possible by accurately and quickly performing a series of skill-based tasks.

      Instructions:
      1. Setup: Place the game board on a flat surface and distribute the game pieces as instructed in the rulebook.
      2. Start: Each player takes turns drawing a task card from the deck. The task card will describe a specific skill challenge (e.g., balancing, stacking, manipulating small objects).
      3. Perform: The player must perform the task within the time limit specified on the card.
      4. Scoring: Points are awarded based on the successful completion of the task and the speed at which it is completed. Refer to the scoring guide for detailed point allocation.
      5. Next Player: Pass the turn to the next player in a clockwise direction.
      6. End of Game: The game ends after a predetermined number of rounds or when all task cards have been drawn. The player with the highest total score wins.

      FAQs:

      Q: What if I fail to complete a task within the time limit?
      A: If you fail to complete a task within the time limit, you do not receive any points for that round. The turn then passes to the next player.

      Q: Can I attempt the same task multiple times?
      A: No, each task can only be attempted once per turn. If you fail, you must wait until your next turn to draw a new task card.

      Q: What happens if two players tie for the highest score?
      A: In the event of a tie, a tiebreaker round is played. Each tied player draws a new task card and attempts the task. The player who completes the task successfully and the fastest wins.

      Q: Are there different difficulty levels for the tasks?
      A: Yes, task cards are categorized into different difficulty levels. Players can decide before the game starts if they want to play with mixed difficulty levels or a specific level.

      Q: Can younger players participate?
      A: Yes, younger players can participate. It is recommended to use the easier task cards for younger or less experienced players to ensure the game is enjoyable for all.

      Q: How long does a typical game last?
      A: The length of the game can vary depending on the number of players and rounds. On average, a game lasts between 30 to 60 minutes.

      Q: Do I need any special equipment to play?
      A: All necessary equipment is included with the game. However, some tasks may require common household items, which will be listed on the task card.

      Q: Can the game be played solo?
      A: While the game is designed for multiple players, it can be adapted for solo play. You can challenge yourself to complete tasks and keep track of your own score.
